This QGIS project published the Hawaii County FADS locations in Leaflet for static mapping delivery through GitHub Pages.

This map serves as a benthic view of the sea floor under each FAD. It overlays the Hawaii Division of Aquatic Resources FADS locations over Global Multi-Resolution Topography (GMRT) Synthesis.

QGIS2Web was used to deliver the FADS GIS in Leaflet and publish to GitHub.
